# Layouts de Notícias para TVs – Digital Signage

Objetivo:
Definir layouts ideais para exibição de notícias nas TVs da plataforma.

O sistema deve suportar **múltiplos formatos de exibição**, dependendo da programação da tela.

---

# Tipos de Layout

O sistema terá três modos principais.

---

# Layout 1 – News Ticker

Barra de notícias estilo canais de televisão.

Exibição:

rodapé da tela

Exemplo:

Últimas Notícias:
Inflação desacelera em fevereiro segundo IBGE • Novo smartphone é anunciado pela Samsung • Exportações brasileiras crescem

Características:

- movimento horizontal
- leitura contínua
- ideal para telas com programação de vídeos

Campos exibidos:

categoria
titulo resumido
fonte

---

# Layout 2 – Card de Notícia

Notícia exibida como card.

Elementos:

imagem
título
resumo
fonte

Layout exemplo:

+-----------------------------+
| imagem da notícia           |
|                             |
| Título da notícia           |
| resumo curto                |
| Fonte                       |
+-----------------------------+

Tempo de exibição:

8 a 12 segundos

---

# Layout 3 – Fullscreen News

Notícia em tela cheia.

Usado como parte da programação da TV.

Elementos:

imagem grande
título
resumo
categoria
fonte

Layout:

+--------------------------------+
| imagem grande                  |
|                                |
| título da notícia              |
| resumo                         |
| fonte                          |
+--------------------------------+

Tempo recomendado:

10 segundos

---

# Layout 4 – Multi News Grid

Tela mostrando várias notícias.

Exemplo:

4 cards simultâneos.

Ideal para:

painéis grandes.

---

# Campos obrigatórios

titulo
summary
imagem
categoria
fonte

---

# Prioridade visual

Maior destaque para:

imagem
titulo
categoria

---

# Otimização para TVs

Limitar:

titulo → 80 caracteres
resumo → 140 caracteres

---

# Transições

Sugestão:

fade
slide
crossfade

---

# Compatibilidade

Layouts devem funcionar em:

TVs
terminais Android
painéis LED
